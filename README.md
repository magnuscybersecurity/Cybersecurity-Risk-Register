

## Overview
A professional cybersecurity risk register identifying threats, vulnerabilities, and mitigation strategies across 16 assets using industry standard risk scoring methodology.

## What's Included
- 16 risks across 14 assets
- Threat and vulnerability analysis
- Inherent and residual risk scoring
- Mitigation strategies for each risk
- Legend and summary tabs

## Frameworks and Standards Referenced
- PCI DSS
- ISO 27001
- NIST Cybersecurity Framework

## Risk Scoring Methodology
- Likelihood and Impact scored 1-5
- Inherent Risk = Likelihood x Impact
- Control Effectiveness scored 1-5
- Residual Risk = Inherent Risk x (Control Effectiveness / 5)

## Assets Covered
- Active Directory
- Admin Accounts
- Backup System
- Company Website
- Customer Database
- Employee Email
- Employee Laptops
- Firewall
- HR Database
- Internal Network
- Payment System
- Source Code Repository
- Vendor Access
- VPN Systems
